# EVYD_PD_Dashboard

产品开发仪表板项目，用于产品开发流程管理和数据可视化。

## 功能特点
- 工作流程管理
- 数据可视化
- 对话型应用接口
- 全局统一使用 Lato 字体

## 项目结构
- `assets/` - 静态资源文件
- `backup/` - 备份文件
- `docs/` - 文档
- `modules/` - 模块代码
- `scripts/` - 脚本文件
- `server/` - 服务器端代码
- `styles/` - 样式文件
- `templates/` - 模板文件

## 最近更新
- 将全站字体统一为 Lato，提供更一致的用户体验
- 优化所有页面的字体加载
- 完善文档

## 安装与使用
请参考相关文档进行安装和配置。

## 字体使用
项目使用 Google Fonts 提供的 Lato 字体，包含以下字重：
- Light (300)
- Regular (400)
- Medium (500)
- Semi-Bold (600)
- Bold (700)
